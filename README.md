## Setup

```bash
# clone github
git clone https://github.com/phoresis92/tmux-config.git

cd tmux-config

# set .tmux.config to home directory
cp .tmux.config $HOME

# remove clone 
cd ..
rm -rf tmux-config
```

OR

```bash
git clone https://github.com/phoresis92/tmux-config.git

cd tmux-config

./install.sh
```
